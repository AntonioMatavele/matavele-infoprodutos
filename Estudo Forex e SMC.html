🚀 Passo a Passo para Publicar seu Site no GitHub Pages

1️⃣ Criar uma Conta no GitHub

Se ainda não tem, acesse GitHub e crie uma conta gratuita.

2️⃣ Criar um Repositório

  1. Após fazer login, clique em "New" (Novo Repositório).
  2. Escolha um nome (exemplo: meu-site).
  3. Selecione "Public" para que seja acessível.
  4. Clique em "Create repository".

3️⃣ Enviar os Arquivos do Site

  1. No repositório criado, clique em "Add file" > "Upload files".
  2. Faça o upload do código HTML que criei para você.
  3. Clique em "Commit changes" para salvar.

4️⃣ Ativar o GitHub Pages

  1. Vá para a aba "Settings" (Configurações).
  2. No menu lateral, clique em "Pages".
  3. Em "Branch", escolha main e clique em "Save".
  4. O GitHub vai gerar uma URL para o site (exemplo: https://seu-usuario.github.io/meu-site).

⚡ Pronto! Em poucos minutos seu site estará online! ⚡
